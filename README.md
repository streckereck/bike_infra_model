# Bike Infrastructure Comfort Classification Model

## Overview
Pipeline to classify bike infrastructure using OSM, Replica, and land use data.

## Workflow
* Define study area
* Prepare reference data
* Join predictors (OSM, Replica, context)
* Build model table
* Train/test split (blocked)
* Fit models
* Evaluate + map outputs

## Notes
- Subset-based development (City of Santa Barbara)
- Will scale to full county
